git 命令(上传)

把网上仓库上传到本地，并对多个人写的内容的更新与自己写的同步

  • git clone 克隆网上仓库到本地
  • git diff 未添加到git缓存区之前详细修改
    查看做了哪些修改。按 q 退出。
  • git add <文件名> 把项目的修改添加到缓存区
  • git commit -m ‘名字’ 更新版本
  • git push 上传网上
  • git pull 多个系统同时更新，先把网上修改的放到本地
  • git log 查看当前版本信息
  • git log –pretty=oneline 显示1行
  • git reset –hard [版本号] 回退到该版本
